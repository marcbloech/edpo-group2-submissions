# Architecture Decision Records

This folder contains concise ADRs for key project decisions.

## ADR List
- 0001: Use Camunda 8 for process orchestration
- 0002: Keep job workers in domain services
- 0003: Split BPMN into signup and payment processes
- 0004: Use Kafka as event ingress to orchestration
- 0005: Standardize runtime on Java 21
- 0006: Add resilience fallbacks for orchestration
